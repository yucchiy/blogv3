# blogv3

My blog generator (v3).
[blog.yucchiy.com](https://blog.yucchiy.com)

## Setup

Use Node.js 16.

```
npm install
```

## Build

Build static files into ./public directory.

```
npm run build
```

## Develop

Run a server for developing.
You can see it on http://localhost:8000

```
npm run develop
```
